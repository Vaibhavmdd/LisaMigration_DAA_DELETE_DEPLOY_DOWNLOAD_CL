StartJ2EE
This subprocess starts the J2EE server by taking the host and the server type.
It starts the server at the specified j2eeHome.
If the host is not localhost, the value for j2eeHome should be set to the location which is available on the host specified.
If j2eeHome is not specified:
If type is tibcohost, then the value of variable TIBCO_HOST_HOME on the appropriate host will be used.
The default type is tibcohost.
The default host is localhost.
Parameter Description
typetype of the server. Currently the following are supported: tibcohost
j2eeHomeJ2EE Server Home Directory. If host is remote, specify the location on the remote host.
timeouttime in seconds to fait for checking the server is started
hostHost on which the J2EE Server should be started

com.tibco.amx

StartAllJ2EE
This subprocess starts the server by taking the environment and the server type.
It then starts all the servers of this type in the given environment.
The list of tibcohosts systems are retrieved from the data file used.
On each system, If the Servertype is tibcohost, the tibcohost located under TIBCO_HOST_HOME will be started through LISA simulator.
Parameter Description
typetype of the server. Currently the following are supported: tibcohost
environmentName of the AMX environment this server belongs
timeouttime in seconds to fait for checking the server is started

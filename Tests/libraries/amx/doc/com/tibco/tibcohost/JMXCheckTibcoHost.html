TIBCO ActiveMatrix Library for LISA

com.tibco.tibcohost

JMXCheckTibcoHost
Returns true if TIBCOHost is up and running.
This sub-process performs a JMX call to the getStatus operation MBean.
It will keep trying till timeout is reached, then return false.

Note: It works with non-secure TIBCOHost only.
Parameter Description
hostTIBCOHost host name
portTIBCOHost JMX port
urlJMX Url. If url is provided, host and port are ignored. Use JMXMP protocol.
timeoutWait for a number of seconds before giving up on TIBCOHost

TIBCO ActiveMatrix Library for LISA

com.tibco.tibcohost

JMXStopTibcoHost
Stops TIBCOHost.
Returns void if shutdown is successful.
This sub-process performs a JMX call to the shutdown operation MBean.

Note: It works with non-secure TIBCOHost only.
Parameter Description
hostTIBCOHost host name
portTIBCOHost JMX port
urlJMX Url. If url is provided, host and port are ignored. Use JMXMP protocol.

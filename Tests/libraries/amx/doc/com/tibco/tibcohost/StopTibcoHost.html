TIBCO ActiveMatrix Library for LISA

com.tibco.tibcohost

StopTibcoHost
Stops TIBCOHost.
This sub-process performs first a tibcohost -stop
It can be executed on a remote TIBCOHost as well as long as there is a LISA simulator running on the remote host.

Note: LISA simulator port must be 2001 on the remote machine.
Parameter Description
hostHost on which to stop the TIBCOHost
tibcoHostinstanceTIBCOHost instance directory
modeStop mode used for stopping the node (normal - immediate - abort). Default is normal.
waitIf set to true the wait becomes synchronous and only returns when all node processes were stopped. Default is false
timeoutMax number of seconds to wait to allow the TIBCOHost to stop

TIBCO ActiveMatrix Library for LISA

com.tibco.tibcohost

StartTibcoHost
Starts TIBCOHost.
This sub-process tries to stop the TIBCOHost first (Mode Abort), clean up the logs, then starts it.
It can be executed on a remote TIBCOHost as well as long as there is a LISA simulator running on the remote host.

Note: LISA simulator port must be 2001 on the remote machine.
Parameter Description
hostHost on which the TIBCOHost should be started
tibcoHostinstanceTibcoHostinstance directory. If host is remote, specify the TIBCOHost instance location on remote host.
timeoutMax number of seconds to wait to allow the TIBCOHost to start

TIBCO ActiveMatrix Library for LISA

com.tibco.tibcohost

StartAllTibcoHosts
Starts all TibcoHosts on all the machines belonging to the specified environment.

This sub-process starts TibcoHosts remotely if necessary (as long as there is a LISA Simulator running on the remote machine).
On each machine, the sub-process tries to kill the tibcohost first, clean up the logs, then start the tibcohost.

Note: Simulators must run on default port 2001.
Note: Simulator name must be set to the hostname where the Simulator is running.
Note: Always run this sub-process from local, do not execute remotely.
Parameter Description
timeoutMax number of seconds to wait to allow the each daemon to start
environmentName of the ActiveMatrix environment

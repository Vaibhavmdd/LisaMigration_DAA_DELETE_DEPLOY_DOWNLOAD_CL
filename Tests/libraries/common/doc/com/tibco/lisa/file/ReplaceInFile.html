com.tibco.lisa.file

ReplaceInFile
Replaces one or more strings globally in a file.
Parameter Description
fileFile where to perform the search-replace
oldStringString to be replaced
newStringNew string value
propertyListName value pairs seperated by comma. ex: name1=value1,name2=value2,.. etc. This can be used in addition to passing property file. Optional
propertyFileNameAbsolute path of the file which contains the property name/value pairs. Optional
prefixprefix to be used for the keys to substitute
suffixsuffix to be used for the keys to substitute
encodingEncoding to use for a file, if blank, the encoding is system default encoding

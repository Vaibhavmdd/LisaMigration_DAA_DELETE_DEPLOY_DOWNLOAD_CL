TIBCO Common Library for LISA

com.tibco.lisa.file

WriteFile
Change the contents of text file in its entirety, overwriting any existing text.
Returns the file path.
Parameter Description
fileFile to write into
contentText content to write in the file
createIfNotExistIf set to true, file is created if it does not exist.
encodingencoding to use for a file, if blank, the encoding is system default encoding
native2asciiIf set to true, converts the content from native encoding to ASCII with escaped Unicode

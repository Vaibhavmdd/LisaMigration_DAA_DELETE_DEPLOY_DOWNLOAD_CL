TIBCO Common Library for LISA

com.tibco.lisa.util

Native2Ascii
Converts a file from native encoding to ASCII with escaped Unicode.
Note that output file is overwritten if it already exists.
Parameter Description
inputFileThe full path of the file to convert
outputFileThe full path of the output file
encodingThe native encoding the input file is in (default is the default encoding for the JVM)
reverseReverse the sense of the conversion, i.e. convert from ASCII to native

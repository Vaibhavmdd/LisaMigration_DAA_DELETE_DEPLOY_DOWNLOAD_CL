TIBCO Common Library for LISA

com.tibco.lisa.util

SetNameValuePairInFile
Set a Name-Value pair in a file.
File can have any extension.
If the Name-Value pair already exists, the current value is replaced by the specified value.
If the Name-Value pair is commented, it gets uncommented and set as specified.
If the Name-Value pair does not exist, it is added at the end of the file.

If you need to pass the character backslash ('\') in the Name-Value pair, escape it by passing '\\'.
Parameter Description
fileFile where to set the Name-Value pair
nameName
valueValue
assignCharCharacter utilized for assignment: equal ('='), white space (' '), ...

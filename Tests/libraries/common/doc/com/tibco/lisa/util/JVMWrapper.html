com.tibco.lisa.util

JVMWrapper
Call the TIBCO Wrapper.
Syntax: <wrapperName> <command> --propFile <propFile> <extraCommand>
where
<wrapperName> the absolute path to the wrapper application. Required
<propFile> the absolute path to the tra file used to launch the wrapper (optional).
<command> can be one of the following command switches (optional):
--install, installs the wrapped application as a windows service
--uninstall, uninstalls a previously installed Windows service
--update, updates a previously installed wrapped application or installs if non-existing
--start, starts a previously installed Windows service
--stop, stops a running Windows service
--run, runs the wrapped application as a console application
<extraCommand> Extra command needed by specific wrapper (optional). For instance BW needs the repository path.

For optional input, use <<null>> as value in order no to pass the input parameter
Parameter Description
wrapperNameAbsolute path to the wrapper application
commandOne of the following command switches: --install, --uninstall, --update, --start, --stop, --run
propFileAbsolute path to the tra file used to launch the wrapper
extraCommandExtra command a specific wrapper might need
startInDirExecute from directory
waitForCompletionWait for the command to complete (true or false)
waitBeforeOnWindowsNumber of seconds to sleep on Windows platforms before starting the JVM wrapper
waitBeforeOnUnixNumber of seconds to sleep on Unix platforms before starting the JVM wrapper
waitAfterOnWindowsNumber of seconds to sleep on Windows platforms after starting the JVM wrapper
waitAfterOnUnixNumber of seconds to sleep on Unix platforms after starting the JVM wrapper

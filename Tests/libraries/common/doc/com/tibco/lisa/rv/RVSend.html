TIBCO Common Library for LISA

com.tibco.lisa.rv

RVSend
Sends TIBCO Rendezvous message(s) on the specified subject
Parameter Description
serviceService
networkNetwork
daemonDaemon
subjectSubject for the message(s)
messagesMessage(s) to be published. Blank space separates messages: value 'message1 message2' will publish two messages. Value '"message1 message2"' will publish one message.

TIBCO Common Library for LISA

com.tibco.lisa.rv

RVStreamSend
Sends TIBCO Rendezvous message(s) on the specified subject with RVStream.

Note2: Supported on Windows only.

RVStream Version: 1.21

Send Options
------------
-file <name> : publish messages from file <name>
-speed <real> : speed-up or slow-down sleep commands (see -usage)
-echo : display messages before publishing
-count : show iterator counter (see -usage)

-rvtype <option> : RVMSG_TYPE, e.g. RVMSG_INT
: rvtype, type, e.g. INT
: singlechar, t, e.g. i
Parameter Description
serviceService
networkNetwork
daemonDaemon
sendOptionsSend options
subjectSubject for the message(s)
